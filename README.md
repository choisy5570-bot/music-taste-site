# 멍설윤의 음악 취향

GitHub Pages에 올릴 수 있는 개인 음악 취향 리포트 사이트입니다.

## 올릴 파일

GitHub 저장소에 아래 파일과 폴더를 그대로 올리면 됩니다.

- `index.html`
- `style.css`
- `script.js`
- `assets/hero-music.png`
- `data/taste.json`

## 내용 수정

`data/taste.json` 안의 아티스트, 태그, 추천곡, YouTube Music 링크를 바꾸면 사이트 내용이 바뀝니다.

Google Takeout 원본 기록은 개인정보가 많을 수 있으니 공개 저장소에 올리지 마세요.
